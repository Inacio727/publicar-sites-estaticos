Exemplos de Comandos

  • ls – Lista arquivos e pastas no diretório atual.
  • ls -la – Lista arquivos, incluindo ocultos, com detalhes.
  • cd [diretório] – Altera para o diretório especificado.
  • cd .. – Volta um nível no sistema de arquivos.
  • cd ~ – Vai para o diretório pessoal do usuário.
  • pwd – Mostra o caminho completo do diretório atual.
  • sudo apt update – Atualiza a lista de pacotes disponíveis.
  • sudo apt upgrade – Instala atualizações disponíveis.
  • sudo apt install [nome_do_pacote] – Instala um pacote.
  • sudo apt remove [nome_do_pacote] – Remove um pacote.
  • sudo apt autoremove – Remove pacotes desnecessários.
